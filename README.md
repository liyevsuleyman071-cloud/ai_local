#Installation
Before running the code, you need to open the terminal and download the gemma2:2b model,which runs locally as shown in the code. The code uses the ollama library.
What you need to type in the terminal:
pip install ollama
ollama run gemma2:2b
ollama pull gemma2:2b

#Usage:
The codes file contains the sample codes needed to run the model as an example. 
If you want to add different code examples, just add the codes in the format in the codes.json file.
